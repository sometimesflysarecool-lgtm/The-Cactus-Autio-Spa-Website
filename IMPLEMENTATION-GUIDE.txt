OVERVIEW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ISSUES FIXED:
✓ Large dark gap/dead space on right side of mobile pages
✓ Page content shifted left and clipped horizontally
✓ Horizontal scrollbar appearing on all phone viewports
✓ Elements exceeding 100% viewport width

ROOT CAUSES IDENTIFIED:
✓ Missing overflow-x: hidden on html/body elements
✓ Fixed padding on navbar/sections (20px-40px) without proper box-sizing
✓ Footer divs with inline styles using calc() widths that may overflow
✓ Images without max-width constraints on mobile
✓ No mobile-specific overflow management at breakpoints

HOW TO APPLY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open Squarespace → Settings → Pages
2. Go to "Website Tools" → "Custom CSS"
3. Copy the ENTIRE contents of: mobile-overflow-fix.css
4. Paste into the Custom CSS box
5. Click "Save"
6. Clear browser cache and test on mobile devices

TESTING CHECKLIST:
□ Open each page on a phone (iPhone X, Android standard sizes)
□ Verify no horizontal scrollbar appears
□ Verify no dark gap on right side
□ Verify all content fits within viewport
□ Test portrait AND landscape orientations
□ Check that desktop view (1200px+) shows no changes

WHAT THE FIX DOES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

GLOBAL FIXES (All devices):
  • Adds overflow-x: hidden to <html> and <body>
  • Ensures box-sizing: border-box on all elements (*,*::before,*::after)
  • Prevents elements from exceeding 100% viewport width

MOBILE FIXES @ max-width: 768px:
  • Navbar padding reduced and constrained (40px → 20px)
  • All sections (.hero, .social-section, etc.) forced to 100% width
  • Full-width sections (nav, header, footer) properly boxed
  • Negative margins zeroed out (if any applied via styles)
  • Images, videos, iframes constrained with max-width: 100%
  • All padding/margin values clipped to prevent overflow
  • Grid layouts switched to mobile-friendly widths
  • Forms and inputs forced to 100% width with box-sizing

SMALL MOBILE FIXES @ max-width: 480px:
  • Navbar padding further reduced (20px → 16px)
  • Section padding tightened (20px → 16px)
  • Button padding optimized for touch targets
  • Extra-tight constraints for very small screens

TABLET FIXES @ max-width: 1024px:
  • Large gaps reduced for intermediate screen sizes
